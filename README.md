🎮 Quiz Valorant

Um quiz interativo inspirado no universo de Valorant, onde você testa seus conhecimentos sobre agentes, mapas e cenário competitivo.

🚀 Funcionalidades

✅ Perguntas sobre Valorant (agentes, mapas e pro players)

🎯 Sistema de pontuação em tempo real

📊 Tela de resultado com feedback baseado no desempenho

🎨 Interface com estilo dark inspirada no jogo

🏆 Efeito visual com ícones de rank caindo na tela final

🧠 Como funciona

O usuário responde uma sequência de perguntas

Cada resposta correta soma pontos

No final, o desempenho define seu "rank":

🔥 Radiante (100%)

💎 Diamante (80%+)

🟣 Platina (60%+)

🟡 Ouro (40%+)

⚙️ Ferro (<40%)
